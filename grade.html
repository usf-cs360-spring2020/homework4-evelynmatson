  • The D-level requirements plus implement one node/link diagram with details-on-demand interactivity in D3v5.
  • ... using the d3.cluster() or d3.tree() layout.
  • The node attribute must be encoded somehow
  • Nodes must have a minimum details-on-demand interactivity showing the node label.
B-Level Functionality
  • The C-level requirements plus implement one space filling diagram with details-on-demand interactivity in D3v5.
  • using d3.partition(), d3.treemap(), or d3.pack() layout.

  • Nodes must have a minimum details-on-demand interactivity showing the node label.

  • The B level requirements plus filtering, brushing, or zooming for one of the visualizations.
  • You only need to implement this interactivity in one of the visualizations, but must still support details-on-demand interactivity in that visualization.
